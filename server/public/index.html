Prompt Library API

A cloud-native API for managing and executing prompt templates

API Endpoints

  • GET /v1/prompts - List all prompts
  • GET /v1/prompts/:id - Get a specific prompt
  • PUT /v1/prompts/:id - Create or update a prompt
  • DELETE /v1/prompts/:id - Delete a prompt
  • PATCH /v1/prompts/:id - Modify a section of a prompt
  • POST /v1/references - Store a text reference
  • GET /v1/references/:id - Retrieve a text reference
  • POST /v1/chat - Chat completion with functions
  • POST /v1/functions - Execute a function directly

Documentation

View the OpenAPI specification: openapi.yaml

For GPT integration, see the AI Plugin manifest: ai-plugin.json

Getting Started

To use this API, you need to:

  1. Include your API key in the x-api-key header with your requests
  2. Use the appropriate content type (application/json)
  3. For streaming responses, handle text/event-stream appropriately
